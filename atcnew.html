bkf2020 ATC: AMC 10/12 Answer Checker (New Version)

What's the difference between this page and the old page?

The difference is that this page has the questions layed out in a grid. This may be harder to use when submitting answers, so this is available along with the older one, so users have a choice.

Link to the old page.

How does it work?

Glad you asked! The AMC 10/12 is a 25 question multiple choice test. This website let's you answer the 25 problems, and then put in the correct answers for those problems. After pressing "submit," you will see your result!

Credits:

  • StackOverflow question that basically set everything up. Thanks Roko C. Buljan!
  • Website that helped with CSS, creating buttons, and so on... Pretty useful for me, at least!
  • YouTube tutorial that changed helped put these questions into a grid!
  • StackOverflow thread that helped set up the grid so that each element had at least a certain width (200px).
  • MDM Web Doc on how to set the width of a grid element to be at least a certain value.
